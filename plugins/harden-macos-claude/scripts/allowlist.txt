# Trusted outbound HTTP domains.
# One pattern per line. `#` starts a comment. Blank lines ignored.
# A leading `*.` means "any subdomain of"; bare hostnames match exactly.
#
# Effect: if EVERY https://… host in a Bash command is allow-listed AND
# there is no sensitive source (id_rsa, env dump, $*_KEY, etc.), the
# `curl|python3` / `curl|node` / `curl|perl` / `curl|ruby` remote-exec
# rule is downgraded from BLOCK to warn.  Piping to `sh|bash|zsh` is
# ALWAYS blocked regardless of domain — compromised TLS / CDN is enough
# reason to never trust downloaded code.
#
# Keep this list narrow.  Add domains only for stable, well-operated APIs
# you genuinely use from shell pipelines.
