# General Purpose Skills

Universal skills that can be applied across any project or domain.

## Available Skills

### [create-claude-agent](./create-claude-agent/)

**Description**: Create production-ready Claude agent projects using the Claude Code SDK. Scaffolds complete project structure with customizable system prompts, tool configurations, and execution framework.

**When to Use**:
- Building autonomous code analysis agents
- Creating code refactoring automation
- Generating documentation agents
- Implementing custom task automation

**Key Features**:
- Complete project scaffolding
- Customizable system prompts
- Tool permission management
- Built-in error handling and logging
- Session metrics tracking
- Extensible skill framework

**Example Use Cases**:
1. **Security Analysis Agent**: Scan codebases for vulnerabilities
2. **Refactoring Agent**: Apply style guidelines and modernize code
3. **Documentation Agent**: Generate API docs from source code
4. **Test Generation Agent**: Create unit tests automatically

## Contributing New Skills

To add a new general-purpose skill:

1. Create a directory: `general-purpose/your-skill-name/`
2. Add `SKILL.md` following the standard template
3. (Optional) Add `scripts/` directory for executable scripts
4. Update this README to list the new skill
5. Submit a pull request

### Criteria for General Purpose Skills

- ✅ Applicable across multiple projects and domains
- ✅ Not dependent on specific technologies (or supports multiple)
- ✅ Solves a common development workflow problem
- ✅ Well-documented with clear examples
- ✅ Tested and production-ready
